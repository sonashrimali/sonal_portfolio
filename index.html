Projects

Myntra Clone (iOS)

This iOS application is a fully functional clone of the popular Myntra e-commerce platform, meticulously developed using SwiftUI to replicate its modern and intuitive user experience. The app allows users to sign up, log in, and manage their personal profiles, featuring smooth authentication workflows and responsive UI design. As part of this project, I was responsible for building the authentication and user profile modules from the ground up, implementing secure login and registration using Firebase Authentication. I also integrated Firebase Firestore to manage real-time backend operations, including user data storage, updates, and profile updates, and seamless session handling. The app is designed to be visually appealing, mobile-friendly, and consistent with contemporary shopping app standards. It includes reusable components, proper state management, and dynamic layouts to ensure scalability and maintainability of the codebase.

Key functionalities:

✅ Firebase-backed user authentication and profile persistence

✅ Reusable UI components built with SwiftUI

✅ Real-time user data updates using Firebase Firestore

✅ Responsive and intuitive interface optimized for iOS devices

✅ Clean project structure for future feature additions (like cart, product pages, etc.)

Tech Stack: SwiftUI, Firebase Authentication, Firebase Firestore, Xcode.

Coffee Shop App (Android)

This native Android application was developed using Java and XML to simulate a fully functional coffee shop ordering system. The app allows users to browse a variety of categorized drink menu, customize orders (such as cup size, milk options, and add-ons), and seamlessly add items to the cart for final checkout. I designed and implemented the entire order flow, from product selection to order confirmation. The UI was structured using XML layouts and follows clean design principles to ensure a smooth and intuitive user experience. Emphasis was placed on activity-to-activity navigation, proper data handling, and maintaining a responsive interface across different screen sizes.

The cart system includes features like:

• Updating quantities

• Removing items

• Real-time price calculation

• Clear visual feedback during order review

• Additionally, the order confirmation screen provides a summary of selected items, pricing, and an estimated preparation time — offering a realistic customer flow experience.

• Firebase Though developed for demonstration purposes, the app is structured to easily integrate Firebase for real-time order storage or authentication.

Key Features:

✅ Dynamic drink menu with images and descriptions

✅ Order customization with add-ons and size options

✅ Interactive cart with real-time updates

✅ Clean UI with activity-based navigation

✅ Scalable codebase using Android Studio best practices

Tech Stack: Android Studio, Java, XML, Firebase (optional for storing orders).

E-Commerce App (Android)

This Android e-commerce application was built using Java and XML and offers a complete online shopping experience. The app features essential functionalities such as product browsing, user authentication, shopping cart management, and order placement.

I implemented a structured MVC architecture to keep the codebase clean, modular, and scalable. The app uses Firebase Firestore as the real-time backend to store and retrieve product data, user information, and cart items. Firebase Authentication is used to securely manage user registration and login processes.

The user interface is designed to be intuitive and responsive, enabling users to:

• Register and log in securely

• Browse products with categories and details

•Add/remove items from the cart

• Place orders with a clean checkout experience

•The app also supports real-time updates and ensures data consistency between views.

Key Highlights:

✅ Firebase Firestore integration for dynamic product & user data

✅ Firebase Authentication for secure user access

✅ Cart functionality with quantity management

✅ Order placement flow with form validation

✅ Follows MVC pattern for maintainability and clean separation of concerns

Tech Stack: Java, Android Studio, Firebase Authentication, Firebase Firestore, Material Design.

Music Player (Web)

This responsive web-based music player was developed using HTML, CSS, and JavaScript, designed to provide users with a clean and interactive audio experience. The player includes essential playback controls such as play, pause, next, and previous, and displays real-time song progress, including the current time and total duration.

I implemented dynamic UI updates using JavaScript DOM manipulation, allowing the UI to reflect the state of the music (e.g., active song highlight, play/pause toggle, progress bar sync). Songs are handled via an array of audio files and metadata, making it easy to update or scale the playlist in the future.

The visual layout was carefully crafted using CSS transitions and animations, providing smooth interactions and a modern, engaging look. The player layout is also responsive, ensuring it looks great on desktops, tablets, and mobile devices.

Key Features: ✅ Play, pause, next, and previous controls with visual feedback

✅ Real-time progress tracking with animated progress bar

✅ Current song metadata (title, artist, cover image) display

✅ Playlist managed via JavaScript array for easy updates

✅ Responsive UI with clean CSS styling and animations

✅ Ideal for exploring DOM events, media APIs, and UI syncing in JavaScript

Tech Stack: HTML, CSS, JavaScript.

Blossom Flower Site (Web)

This project is a beautiful, responsive static website for a fictional flower shop, designed and developed using pure HTML and CSS. The goal of the project was to create an aesthetically pleasing and user-friendly interface that showcases various flower arrangements and services in a visually engaging format.

The site includes essential sections such as:

• Home – Featuring a hero banner with a background image and introductory call-to-action

• About Us – Describing the shop’s story and values

• Shop – Displaying featured flower collections and categories

• Contact – With a styled contact form and contact information layout

Special attention was given to the color palette, typography, and layout structure to reflect the elegance and delicacy associated with floral design. The design is fully responsive, adjusting gracefully across different screen sizes and devices, ensuring a consistent experience for both desktop and mobile users.

Animations and hover effects were also incorporated to enhance interactivity and make the browsing experience smooth and modern — all without using JavaScript.

Key Highlights: ✅ Clean, semantic HTML structure

✅ Responsive CSS grid and flexbox layout

✅ Stylish color combinations and custom typography

✅ Animated buttons and hover effects

✅ Fully functional contact form layout (structure-ready for backend integration)

✅ Ideal for showcasing creativity in static web design and front-end layout practices

Tech Stack: HTML, CSS, Responsive Design.
